## 프로젝트 개요

클로드코드 나잇(클코나잇) 시즌 2 발표 신청 및 발표 자료 준비 프로젝트.

발표자: 팀스파르타 모바일 게임 마케팅 담당 (비개발자)
발표 시간: 15분
발표 주제: **내 업무 하나를 통째로 에이전트에게 넘겨본 이야기**

핵심 메시지: *AI 활용의 병목은 기술이 아니라 문제 정의 능력이다.*

## 발표 콘텐츠 구조

**신청서 파일**: `클코나잇_신청서.md` — 발표 주제, 자기소개, 내용 소개 포함.
**행사 소개**: `클코나잇_행사소개.md` — 행사 개요, 환영 주제, 일정, 혜택 포함.
**포트폴리오**: `포트폴리오.md` — 자동화 13개 프로젝트별 목적/로직/스택/절감 시간 정리.

발표 3축:
1. **성과**: 13개 자동화 제작, 9개 팀 전체가 매일 사용 (CS·마케팅·사업·기타)
2. **실패담**: 파일명 규칙 부재로 소재 오분류, Meta API rate limit 미숙지, "나만 쓰는" 도구 문제
3. **인사이트**: 도메인 지식(판단 기준)이 곧 자동화 설계도 → 논리 구조가 먼저 정리돼야 함

청중 takeaway: *"나는 매일 어떤 판단을 반복하고 있지? 그게 규칙으로 쓰일 수 있나?"*

1. 이벤트 소개 자료 추가 시 → 발표 내용과 청중 맞춤 조정

3. 발표 내용 피드백 요청 시 → 비개발자 청중 눈높이 기준으로 검토
